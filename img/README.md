# Папка для изображений

Здесь храните все изображения для вашего проекта.

## Рекомендуемая структура:

```
img/
├── hero/           # Изображения для главного экрана
├── about/          # Изображения для раздела "О нас"
├── services/       # Изображения для услуг
├── team/           # Фотографии команды
├── clients/        # Логотипы клиентов
├── gallery/        # Галерея проектов
└── icons/          # Иконки
```

## Рекомендуемые форматы:
- **JPG/JPEG** - для фотографий
- **PNG** - для логотипов, иконок (с прозрачностью)
- **SVG** - для иконок и логотипов (векторные)
- **WebP** - современный формат для веба

## Оптимизация:
- Сжимайте изображения перед загрузкой
- Используйте правильные размеры
- Для веба: 72 dpi
